# Azkaar - Islamic Supplications App

A premium Flutter application for daily Islamic Azkar, featuring a modern "Shorts-style" vertical scroll interface and a beautiful Islamic aesthetic.

## Features

- **Premium Islamic Design**: Dark emerald theme with gold accents and high-quality Islamic geometric pattern background.
- **Social Media Style Scroll**: Vertical PageView for a seamless browsing experience.
- **Glassmorphism UI**: Frosted-glass style cards for maximum readability and a premium feel.
- **Multilingual Support**: Available in Arabic, English, French, German, Japanese, and Chinese.
- **Customizable Experience**:
  - Adjust font size dynamically (persists across sessions).
  - Toggle between "Counter Mode" and "Scroll Mode" (default).
- **Progress Tracking**: Real-time progress visualization using a sleek pie chart.
- **Dynamic Counter**: Animated circle counter with haptic-like visual feedback.
- **Automatic Reset**: Returning to the start of the list resets your progress for a new session.

## Screenshots

## Build and Installation

### Web
To build for web:
```bash
flutter build web
```

### Android
To build the APK:
```bash
flutter build apk --release
```
